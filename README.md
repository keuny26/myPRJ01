# 🧠 파이썬 MBTI 간이 검사 프로그램

## 📜 프로젝트 소개

이 프로젝트는 **파이썬**을 사용하여 개발된 간단한 MBTI(Myers-Briggs Type Indicator) 간이 검사 프로그램입니다. 
사용자에게 12가지 질문을 제시하고, 응답을 바탕으로 4가지 MBTI 지표를 측정하여 최종 유형을 도출하며, 간략한 해설을 제공합니다.

이는 정식 MBTI 검사가 아니며, 재미로 자신의 성향을 가늠해 보는 데 초점을 맞추고 있습니다.

## 🚀 사용 방법

### 1. 프로그램 실행

터미널(Terminal) 또는 명령 프롬프트(Command Prompt)를 열고, 프로그램 파일(`.py`)이 위치한 디렉토리로 이동한 후 다음 명령어를 입력하여 실행합니다.

```bash
python mbti_test.py
